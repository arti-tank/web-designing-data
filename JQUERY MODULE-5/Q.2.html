1.how to apply CSS using jQuery,how to add class and remove class in jquery,jQuery animation?

  • css apply:sets the style attribute:-

  • EXAMPLE:-

    $(document).ready(function()
    { $('button').click(function()
    { $(p).css('color,blue'); } ); } );

  • add a class:a one or more to the select element.
  • simply to appy in jquery mathod and add to select
    element then run to the browser.

    • EXAMPLE:-

    $(document).ready(function()
    { $( 'button').click(function()
    { $(p).addClass( 'color,blue'); } ); } );

  • remove class:
    same condition to add class but different beetween add class is
    more class add when remove class is more class remove this file.

  • EXAMPLE:-

    $(document).ready(function()
    { $( 'button').click(function()
    { $(p).removeClass( 'a'); }); } );


  • add to animation:
    The jQuery animate mathod ('') is used to make a custom animation.
    animation is most important part of jquery.
    then step by step apply to css property in animation
    and creat a beutyful animation.

  • EXAMPLE:-

    $(document).ready(function()
    { $('button').click(function()
    { $(p).animate('a'); }); } );
